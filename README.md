# Njan Kanda Cinema — Releases

Public binary distribution channel for the [Njan Kanda Cinema](https://www.njankandacinema.in)
Android app.

This repository only hosts the signed APKs. The application source code
lives in a separate, private repository.

## Install

**[⬇️ Download the latest APK](https://github.com/deepak-delta/Njan-Kanda-Cinema-Releases/releases/latest/download/app-release.apk)**

On first install, Android will ask you to allow installs from your browser
— tap the prompt and the install completes. Sign in with the same account
you use on the web app.

## All releases

See the [Releases](https://github.com/deepak-delta/Njan-Kanda-Cinema-Releases/releases)
tab for the full version history, changelogs, and downloads.

## About the app

Njan Kanda Cinema is a private, Letterboxd-style movie diary — log films,
rate them with half-star precision, capture where you watched them, keep
a watchlist, and watch your taste take shape over time.

The web version is at **https://www.njankandacinema.in** and shares the
same account as the Android app.
